# User Stories

## Caregiver Stories
- As a caregiver, I want to view upcoming tasks and appointments in one place so I can stay organized.
- As a caregiver, I want to receive reminders so I do not miss important care activities.
- As a caregiver, I want to share updates with the care team so everyone stays informed.

## Patient Stories
- As a patient, I want to see my upcoming care activities so I can prepare for them.
- As a patient, I want to access support information easily so I can manage my care confidently.

## Provider Stories
- As a provider, I want to review care coordination updates so I can support the patient effectively.

# Epic: User Management

User Story:

As a patient, I want to create an account so that I can securely access my healthcare information.

Acceptance Criteria
User can register with:
First name
Last name
Email
Password
Email must be unique.
Password must be securely hashed.
Validation errors are returned for invalid input.
A successful registration returns the new user's details (excluding the password).
